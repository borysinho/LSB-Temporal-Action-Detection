#    python -c "import torch; print(f'PyTorch: {torch.__version__}')"
#    python -c "import torch; print(f'CUDA: {torch.cuda.is_available()}')"
#    python -c "import transformers; print(f'Transformers: {transformers.__version__}')"
#
# 4. Requisitos de Hardware:
#    - GPU: NVIDIA con ≥12GB VRAM (recomendado: ≥24GB)
#    - RAM: ≥32GB (recomendado: ≥64GB)
#    - Storage: ≥100GB SSD
#
# 5. Modelos Pre-entrenados:
#    Los modelos se descargarán automáticamente de Hugging Face:
#    - Video Swin Transformer: ~400MB
#    - TimeSformer: ~350MB
#    Total: ~5-10GB con cache
#
# 6. Alternativas ligeras:
#    Si tienes GPU más pequeña (<12GB), considera:
#    - Reducir batch_size en config.yaml
#    - Usar backbone 'slowfast' en vez de 'video_swin'
#    - Reducir clip_length a 32 frames
#
# 7. CPU-only (NO RECOMENDADO para este sistema):
